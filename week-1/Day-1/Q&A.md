**1. What problem does Salesforce solve?**



Salesforce solves the problem of managing customer information and business activities in different places. In many companies, customer details, sales records, emails, and support information are stored separately, which creates confusion and reduces productivity. Salesforce provides a single cloud-based platform where businesses can store and manage all customer-related data efficiently.



It helps organizations:



* Track customer interactions
* Manage sales and leads
* Improve customer support
* Automate business processes
* Generate reports and analytics



By using Salesforce, companies can improve communication between teams, increase sales, provide better customer service, and make faster business decisions. Since it is cloud-based, employees can access data from anywhere using the internet.





**2. What is CRM?**



CRM stands for Customer Relationship Management. It is a technology or software system used by businesses to manage relationships and interactions with customers.



A CRM system helps companies:



* Store customer information
* Track sales and leads
* Record customer communication
* Manage customer support requests
* Improve customer satisfaction



The main goal of CRM is to build strong relationships with customers and improve business growth. Instead of storing customer details in spreadsheets or notebooks, CRM keeps everything organized in one centralized system.



Salesforce is one of the most popular CRM platforms in the world because it provides tools for sales, marketing, customer service, and automation.



**3. What is an Object in Salesforce? Give one example**.



In Salesforce, an Object is similar to a database table that stores specific types of information. Objects contain records and fields.



* Fields are individual pieces of data.
* Records are complete sets of information stored in the object.



Salesforce has two types of objects:



1. Standard Objects – Already provided by Salesforce
2. Custom Objects – Created by users based on business needs

Example:



The Account Object is a standard object in Salesforce.



It stores information about companies or customers such as:



* Company Name
* Phone Number
* Address
* Industry Type



For example, if a company wants to store details about its clients, it can use the Account Object to maintain all the related information in one place.



**4. Difference between Salesforce Admin and Developer**

&#x20;  

|Salesforce Admin|Salesforce Developer|
|-|-|
|A Salesforce Admin manages and configures the Salesforce platform.|A Salesforce Developer builds custom applications and features on Salesforce.|
|Mostly works with no-code or low-code tools.|Works mainly with coding and programming.|
|Creates users, roles, reports, dashboards, and workflows.|Develops custom logic using Apex and Lightning Components.|
|Responsible for security settings and data management.|Responsible for advanced customization and integrations.|
|Helps automate business processes using tools like Flow.|Writes code for complex automation and business requirements.|
|Requires platform knowledge and configuration skills.|Requires programming knowledge along with Salesforce concepts.|





In simple words:

* Admin manages and maintains Salesforce.
* Developer customizes and builds new features in Salesforce.



Both roles are important for implementing Salesforce successfully in an organization.





**5. Suggest one real-world application that can be built using Salesforce.**



One real-world application that can be built using Salesforce is a Hospital Management System.



This application can help hospitals manage:

* Patient records
* Doctor information
* Appointments
* Billing details
* Medical history
* Customer support

Features of the system:

* Patients can book appointments online.
* Doctors can access patient history easily.
* Automated reminders can be sent for appointments.
* Hospital staff can track billing and payments.
* Reports can be generated for hospital management.



Using Salesforce for this system improves efficiency, reduces manual work, and provides better service to patients. Since Salesforce is cloud-based, hospital data can be accessed securely from different locations.
